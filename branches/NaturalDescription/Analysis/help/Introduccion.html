Introduccion a Algoritmos

Un algoritmo es un conjunto de instrucciones que conducen a la solución de un problema o situación por resolver. Un algoritmo expresa un procedimiento paso a paso para solucionar un problema dado. Por ejemplo, supongamos que deseamos hacer una llamada telefónica. Elaboremos una lista de pasos, para que podamos hacerla, digamos algo similar a lo siguiente: 1. Tomar el teléfono, 2. Marcar el número telefónico, 3. Esperar a que contesten, 4. Platicar un rato y 5. Terminar la llamada. De esta manera, tenemos una forma ordenada de realizar, en este caso, una llamada telefónica. Entonces, tenemos un algoritmo para tal fin. Sin embargo, los algoritmos pueden considerar diferentes situaciones que pudieran presentarse

Las características que debe cumplir un algoritmo son ser:

  1. Preciso, para indicar el orden de realización de cada paso.
  2. Definido, de manera que si un algoritmo se sigue dos veces, se debe obtener el mismo resultado cada vez.
  3. Finito, es decir, debe tener un número finito de pasos.

Un algoritmo debe de constar de tres partes:

  1. Entrada, que es la información que se le proporciona al algoritmo.
  2. Proceso, que es la realización de las operaciones necesarias para producir la salida.
  3. Salida, que es la información producida por el algoritmo.

Diagramas de Flujo
Los diagramas de flujo son una herramienta para la representación de algoritmos, en la cual utiliza símbolos que están unidos por flechas. Cada símbolo tiene un significado y las flechas, denominadas líneas de flujo, indican la secuencia que tienen los símbolos. Cada uno de los símbolos de los diagramas de flujo indican el tipo de operación a realizar y el diagrama de flujo muestra gráficamente la secuencia en la que se realizarán las operaciones. Las líneas de flujo representan el flujo secuencial de la lógica del algoritmo. Un rectángulo () significa algún tipo de proceso, es decir, acciones a realizar. El paralelogramo es un símbolo de entrada/salida que representa cualquier tipo de entrada o salida del algoritmo. El símbolo rombo es una caja de decisión que representa respuestas sí/no o bien diferentes alternativas. Un símbolo terminal se usa en los diagramas de flujo para indicar su comienzo y terminación. Un pequeño círculo es un conector y se utiliza para conectar caminos, tras roturas previas en el flujo del algoritmo.
